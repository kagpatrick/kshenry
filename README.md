# Henry Sohre Kitiabi, PhD, MPH — Static Academic Site

This is a lightweight static website (HTML/CSS/JS) modeled after AcademicPages.
It is ready for **GitHub Pages**.

## Quick Deploy (GitHub Pages)

1. Create a new repository on GitHub (e.g., `kitiabi-site`).
2. Upload these files (or push via Git).
3. In the repo settings → **Pages**, set the branch to `main` (root).
4. Visit the public URL shown by GitHub.

## Customize

- Replace `assets/img/profile.txt` with an actual profile image (e.g., `profile.jpg`).
- Replace link placeholders in `contact.html` and the sidebar in the template.
- Swap `assets/CV.pdf` with your real CV.

## Colors & Fonts

- Accent: steel blue (#3A6EA5), text: #1F2937, muted: #4F4F4F
